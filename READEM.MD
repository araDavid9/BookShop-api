Instruction Files to run this project :
----

1. First you need to run the backend so please make sure your .NET SDK is >= 8.0.0
2. Open the terminal , go into the backend folder and insert : dotnet restore
3. run dotnet run 

now the server should run on http:localhost:5000 
** this project is running localy so i chose to do it with http

now run the client side 

make sure your node version is >= 24.7.0
1. open terminal and go to the client folder
2. run npm install
3. than create .env file : VITE_API_URL = http://localhost:5000/api (or any other port you chose for the server)
4. npm run dev
